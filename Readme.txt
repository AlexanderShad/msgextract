Простенький скрипт автоматизации распаковки файла сообщения электронной почты MSG, чтоб можно было делать одной командой, а не двумя.

Ну это при условии, что если нужны именно файлы внутри сообщения, а не eml.

Использует в работе пакеты
- perl-Email-Outlook-Message-scripts 
- mpack

Делался под ALT Linux

Spec для сборки в пакет приложен

Для установки на ALT Linux без пакета, либо с неофициального пакета, перейдите в раздел Releases
